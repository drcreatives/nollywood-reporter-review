- **Review Title (H1)**: Montserrat Bold, 40px (2.5rem)
- **Subheading (H2)**: Raleway Medium, 28px (1.75rem)
- **Body Text**: Lora Regular, 18px (1.125rem)
- **Meta Info**: Poppins Light, 14px (0.875rem)
- **Pull Quotes**: Lora Italic, 20px (1.25rem)
